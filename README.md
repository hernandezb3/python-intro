# Introduction to Python
This repository contains a collection of introductory Python programming exercises, assignments, and Jupyter notebooks completed while learning the fundamentals of Python. 

The repository covers a variety of core programming concepts, including:
- Variables and data types
- Arithmetic and logical operators
- Strings and string manipulation
- Conditional statements
- Loops and iteration
- Functions
- Lists, tuples, dictionaries, and sets
- File input/output
- Basic object-oriented programming concepts
- Debugging and code organization

## Purpose
The goal of this repository is to build a foundation in Python programming while demonstrating the ability to write clean, readable, and well-documented code. Each notebook and assignment focuses on applying fundamental concepts to solve programming problems and reinforce computational thinking.


## Technologies
- VS Code
- Python 3
- Jupyter Notebook
